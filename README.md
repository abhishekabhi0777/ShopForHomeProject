# ShopForHomeProject

## Project Description
ShopForHome is a full stack e-commerce web application developed using Angular for frontend and .NET Core Web API for backend. It allows users to browse products, add items to cart, and manage orders.

## Technologies Used
Frontend: Angular  
Backend: .NET Core Web API  
Database: SQL Server  
Tools: Git, GitHub, Visual Studio, VS Code

## Project Structure

Frontend_ShopForHome/
Database_Schema_ShopForHome

## Features
User Registration and Login
Product Browsing
Add to Cart
Wishlist Management
Order Placement
Admin Product Management
